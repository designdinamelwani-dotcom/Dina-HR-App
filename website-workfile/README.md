# Website Workfile

A portable static website project that works on any laptop or system.

## How to use

1. Open the `website-workfile` folder.
2. Double-click `index.html`, or open it in any browser.
3. Edit `index.html`, `style.css`, and `script.js` to customize the site.

## Notes

- No build tools or package managers are required.
- Works locally on macOS, Windows, Linux, and most browsers.
- If you want a live reload workflow, open the folder in a simple web server such as Python:

  ```bash
  cd website-workfile
  python3 -m http.server 8080
  ```

- Then visit `http://localhost:8080`.
